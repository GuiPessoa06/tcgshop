## 🎯 Objetivo do Software
O sistema tem como objetivo simular uma loja de cartas (TCG), permitindo o cadastro de cartas, adição ao carrinho, realização de pedidos e visualização dos pedidos realizados.

---

## ⚙️ Funcionalidades do Sistema

### 📦 Cartas
- Cadastrar novas cartas
- Listar cartas cadastradas

### 🛒 Carrinho
- Adicionar itens ao carrinho
- Visualizar itens do carrinho
- Calcular total da compra

### 💳 Pedido
- Finalizar pedido
- Salvar pedido no banco de dados

### 📋 Pedidos
- Listar pedidos realizados
- Visualizar itens de cada pedido

---
